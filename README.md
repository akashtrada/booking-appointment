# Booking Appointment

A React application built with TypeScript and Material UI (MUI).

## Tech Stack

- **React 18** — UI library
- **TypeScript** — Type safety
- **MUI v6** — Material Design component library
- **Vite** — Build tool and dev server

## Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
